# PACKET SNIFFER

Hi all! This is a packet sniffer in python.

## Libraries
I have used ```scapy``` library

## Getting started
1. First of all - 
```
pip install requirements.txt
```

2. Secondly - 
```
python main.py
```
OR
```
sudo python3 main.py
```

That's all. This is how we can use the packet sniffer.
